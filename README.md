## ğŸš€ Getting Started

### Prerequisites

- Node.js (v18+)
- Python (v3.8+)
- pip

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/yourusername/resumatch-ai.git
    cd resumatch-ai
    ```

2.  **Setup Backend**
    ```bash
    cd server
    pip install -r requirements.txt
    ```

3.  **Setup Frontend**
    ```bash
    cd ../client
    npm install
    ```

### Running the Application

1.  **Start the Backend Server**
    ```bash
    # In /server directory
    python app.py
    ```
    The server will start on `http://localhost:5000`.

2.  **Start the Frontend Client**
    ```bash
    # In /client directory
    npm run dev
    ```
    The client will start on `http://localhost:5173`.

3.  Open your browser and navigate to the client URL!
